In this assignment, I used JavaScript async/await and a try...catch block to cleanly fetch and handle live data from the City of Winnipeg Open Data API. I parsed the JSON responses using object destructuring with default fallbacks to ensure the tree metadata renders smoothly to the DOM even if some fields are missing from the server. The trickiest part was handling search inputs safely, which I solved by using Socrata Query Language (SoQL) filters and encodeURI() to keep spaces from breaking the network fetch. Finally, I strictly followed the mandatory version control guidelines by working entirely on an isolated feature branch with over 10 meaningful commits, making it easy to trace my code logic in the browser debugger.
